# PrepNexa

AI-powered exam preparation platform with PDF learning, quiz generation, weak-topic analysis, and personalized study assistance.

---

## Features

- 📄 Upload PDFs for learning
- 🤖 AI chatbot for doubt solving
- 📝 Automatic quiz generation
- 📊 Weak topic detection
- 🎯 Personalized study assistance

---

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python
- FastAPI

### AI Integration
- Gemini API

## Installation

### Clone Repository

```bash
git clone https://github.com/saroj-mahata/PrepNexa.git
```

### Backend Setup

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend Setup

Open the frontend files in browser or run locally.
